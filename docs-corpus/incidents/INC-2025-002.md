# INC-2025-002 — Latency spike from slow downstream

## Summary
p95 latency crossed 2s for 5 minutes; error rate stayed low but requests were
slow enough to trip the alert.

## Root cause
A downstream provider degraded; no app change was involved. The service itself
was healthy.

## Resolution
No revert applicable. Enabled the cached fallback path and opened a ticket with
the provider. Confidence that this was app-caused was low, so the agent
escalated instead of acting — the correct call (see RB-05).

## Duration
5m. No availability loss; correctness intact.
